# Coloring_Server

Coloring is the use of artificial intelligence to filter color recommendations and photos.  
  
컬러링은 인공지능을 활용하여 색상을 추천해주고 사진을 필터링해주는 기능입니다. 

# how to use

```
#use socket
python server.py
#use bluetooth
python server_blue.py
```
